### 作品緣由
家裡經營早餐店，在忙碌時希望客人可以手寫點餐，有些客人還是習慣直接用說的點餐。在忙碌時會不小心遺漏，以導致客訴。<br/>
希望可以藉由這樣的點餐系統產生，讓客人可以更方便的點餐，也提升櫃台的出餐效率。<br/>

## 網站功能
### 顧客
- 瀏覽最新店內消息公告
- 查看店內販售商品
- 線上點餐、送出訂單
- 利用「簡單點」，解決每天早晨想吃什麼的困擾
 
### 店家
- 依據後臺訂單，可以看最新訂單狀
- 透過後臺編修

</br>

## 專案技術
- Vite 環境建置
- Vue Router 建構路由
- Pinia 全局狀態管理
- 元件拆分與使用
- Vue axios 串接 RESTful API 資料
- Bootstrap 完成 RWD 版型
- Vuelidate 表單驗證
